chicagocrime <- tempfile()
chicagocrime <- read.csv("Crimes_-_One_year_prior_to_present.csv")

Who: Data is collected by the Chicago Police Department through their CLEAR (Citizen Law Enforcement Analysis and Reporting) system

What: This data reflects reported incidents of crime that have occurred in the City of Chicago during a specific time period. This data set contains 90 days of information and the most recent data available is seven days prior to todays date. In order to protect the privacy of crime victims, addresses are shown at the block level only and specific locations are not identified.

Limitations: This data is incredibly recent and is updated daily. It could be cool to create a live visualization with the daily updated data. Limitations include that there isnt exact locations of the data, so thinking about exact proximity to a transit station could be somewhat difficult

chicagocrime <- chicagocrime %>%  
  mutate(Date = mdy_hms(DATE..OF.OCCURRENCE)) %>% 
  mutate(DateTime = force_tz(Date, "America/Chicago")) %>% 
  mutate(year = year(Date)) %>% 
  mutate(month = month(Date))

latestcrime <- chicagocrime %>% 
  filter(year == 2023) %>% 
  filter(month == 9)

street <- latestcrime %>% 
  filter(LOCATION.DESCRIPTION == "STREET") %>% 
  mutate(hour = hour(Date)) %>% 
  filter(hour >= 22)
ggplot(street, aes(x=PRIMARY.DESCRIPTION)) +
  geom_bar() +
  scale_x_discrete(guide = guide_axis(angle = 70)) +
  xlab("Type of Crime") +
  ylab("Number of Occurrances")

londonbus <- tempfile()
londonbus <- read.csv("injury-details-2023-q2.csv")

incidenttype <- londonbus %>% 
  group_by(Incident.type) %>% 
  tally()

londonbus <- filter(londonbus, Incident.type %in% c("Assault", "Robbery", "Vandalism / Hooliganism"))

londonbus <- filter(londonbus, Gender != "Unknown")

Who: TFL (Transport for London) through their Bus Safety Dashboard

What: All bus related injuries in TFL buses since 2014, filtered to only include injuries caused by other individuals

Limitations: Doesnt provide specific location of accidents, though it provides international data. Also focuses more on incidents occurring on buses so doesnt look at data in surrounding area
